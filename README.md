# SwiftWeatherApp  
A weather app built with Swift that uses an API to display current weather conditions for any city.  

## Features  
- Fetch weather data.  
- Displays temperature, humidity, and weather description.  
- Simple and clean UI.  

## Requirements  
- iOS 14+  
- Xcode 13+
